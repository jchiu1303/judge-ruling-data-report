# Federal immigration bond cases by judge

Interactive report of federal district court immigration bond / habeas corpus dockets
in **California, New York, and Texas** (filed 2016+), with grant/deny labels where
RECAP docket snippets allow parsing.

## Methodology (short)

Immigration bond outcomes are derived from [CourtListener](https://www.courtlistener.com/)
RECAP docket snippets. "Not parsed" cases lack grant/deny language in the available text
and are excluded from grant-rate denominators unless otherwise noted. Immigration bond
is distinct from state criminal bail.
